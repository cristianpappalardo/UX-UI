# Responsive Menu B

This example uses **min-width** media queries only.

## Behavior

- Default (mobile): vertical stacked menu.
- At `min-width: 700px`: menu turns into a horizontal, wrapping row.
- At `min-width: 1024px`: container width is centered and menu spacing increases.

## Notes

- Mobile-first structure and styles.
- Semantic structure with `<header>`, `<nav>`, `<ul>`, `<li>`, `<a>`, and `<main>`.
- Includes 8 menu items.
