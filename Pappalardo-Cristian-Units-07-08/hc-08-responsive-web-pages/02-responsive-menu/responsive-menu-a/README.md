# Responsive Menu A

This example uses **max-width** media queries only.

## Behavior

- Default (desktop): horizontal navigation bar.
- At `max-width: 768px`: menu wraps across multiple rows.
- At `max-width: 520px`: menu becomes a vertical stack.
- At `max-width: 340px`: heading scales down for tiny screens.

## Notes

- Semantic structure with `<header>`, `<nav>`, `<ul>`, `<li>`, `<a>`, and `<main>`.
- Includes 8 menu items.
