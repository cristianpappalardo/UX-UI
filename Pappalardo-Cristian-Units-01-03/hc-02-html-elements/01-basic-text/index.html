Studio Notes

Welcome to a small collection of ideas, experiments, and tiny wins from the week. Each entry is short on purpose so the core takeaway stays clear.

I use this page as a simple habit tracker and a reminder to finish what I start before chasing the next shiny thing.

Current Focus

This month is about rhythm: consistent practice, steady feedback, and clean execution. The goal is to build a reliable routine rather than chasing perfect results.

Small daily tasks are more useful than big plans, so I list one priority and two supporting actions.

Design Warmups

Warmups begin with a single constraint: one typeface, one color, one layout idea. The constraint keeps the work honest and focused.

After a quick sketch, I write a one-sentence critique and underline the one improvement to try tomorrow.

Micro Projects

Micro projects are small enough to finish in a day, like a landing page or a profile card. Shipping fast makes the feedback loop surprisingly fun.

Each project ends with a short checklist: what worked, what broke, and what felt unnecessarily hard.

Tools and Tactics

My toolkit is simple: browser dev tools, a text editor, and a notebook. The best tool is still the habit of writing clear notes.

When I get stuck, I step back and reframe the problem in one plain sentence.
